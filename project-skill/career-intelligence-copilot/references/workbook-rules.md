# Workbook Rules

## Positioning

The workbook should function as a decision board.

It should usually contain sections such as:

- scoring framework
- market conclusions
- personal positioning
- company strategy
- ranked companies
- recruiter and interview questions

## Ranking Logic

Useful dimensions often include:

- compensation upside
- base monthly pay
- company moat
- resume value
- skill growth
- stability
- workload friendliness
- personal fit

## Personal Positioning

This section should answer:

- what roles the user best fits now
- what they are missing for the next step up
- how the resume should change

## Company Ranking

For each company, prefer maintaining:

- company name
- representative role
- role characteristics
- key responsibilities
- pay judgment
- recommendation reason
- main risks

## Rule Of Thumb

If the project has real sampled jobs, company ranking should be evidence-based rather than purely imaginative.
