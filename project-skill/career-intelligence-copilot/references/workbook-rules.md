# Workbook Rules

The workbook is a decision board. It should help the user compare companies and roles quickly.

It is not the right place to put raw long-text fields in the first columns.

## Recommended Sheets

- `Scoring Framework`
- `Market Conclusions`
- `Personal Positioning`
- `Company Strategy`
- `Companies`
- `Recruiter Questions`

Localized sheet names are fine. Keep the responsibilities clear.

## Scoring Framework

Recommended dimensions:

- ownership
- base location
- work intensity
- total-pay upside
- monthly salary
- assets and moat
- resume value
- skill growth
- career stability
- overtime friendliness
- personal fit

## Market Conclusions

Cover:

- market structure
- salary center
- high-pay sources
- company type
- ownership
- base location
- work intensity
- platform differences
- company scale

Convert descriptive findings into decision-ready signals:

- monthly salary
- bonus months
- total-pay certainty
- whether work intensity offsets pay
- whether location or on-site requirements reduce attractiveness

## Personal Positioning

Answer:

- what roles the user currently fits best
- what capability tags are missing
- how the resume should change
- what portfolio projects would be most useful

These conclusions must come from real accepted postings.

## Company Strategy

Summarize company buckets:

- state-owned / private / foreign / joint venture / unclear
- heavy-asset / light-asset
- asset side / user side / platform / aggregation / retail
- base location and remote/on-site expectations
- high-value role tags
- core upside
- risks

## Companies

Only include companies from the accepted master dataset.

Recommended fields:

- rank
- company name
- representative role
- ownership
- business side
- base location
- Beijing / local / remote attribute, if relevant
- work-intensity judgment
- travel or on-site requirement
- role characteristics
- responsibility summary
- category
- tier
- asset weight
- premium tags
- market evidence
- recommendation line
- suitable role
- pay judgment
- scoring dimensions
- total score
- recommendation level
- recommendation reason
- main risks

Display order:

- short decision fields first
- long explanation fields later

Do not place long responsibility text near the front of the sheet.

## Recruiter Questions

Questions should target decision uncertainty:

- bonus mechanism
- trading authority
- headcount ownership
- reporting line
- base location
- asset ownership
- technical maturity
- team maturity
- work rhythm

## Update Order

1. Master dataset.
2. Market conclusions.
3. Personal positioning.
4. Company strategy.
5. Company ranking.
6. Recruiter questions.
