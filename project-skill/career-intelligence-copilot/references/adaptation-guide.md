# Adaptation Guide

## Fastest Way To Adapt This Skill

When reusing this skill in a new project, update these four things first:

1. dataset paths
2. workbook paths
3. target platforms
4. filtering rules

## Example Adaptations

### Domain-specific market

Examples:

- power trading
- biotech BD
- SaaS product manager
- AI engineering

In this case, keep the workflow and only adapt:

- search keywords
- company taxonomy
- ranking dimensions
- representative role logic

### Broad job search

If the user wants many functions or industries at once:

- split the dataset by role family
- keep one shared workbook summary
- compare pay and demand across clusters

## Public vs Private Repo

Recommended pattern:

- public repo for the reusable skill
- private repo for live data and personal materials
