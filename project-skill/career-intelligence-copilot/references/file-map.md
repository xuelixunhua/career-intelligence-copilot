# File Map

This repository is designed to be adapted into real workspaces.

## Typical Layers

### Skill Layer

- `project-skill/career-intelligence-copilot/SKILL.md`
  - the workflow entry point

- `project-skill/career-intelligence-copilot/references/*.md`
  - support rules for analysis and maintenance

### Data Layer

Expected in the consuming project, not necessarily in this public repo:

- master dataset json
- master dataset csv
- workbook or spreadsheet
- resume PDFs

### Local Runtime Layer

Expected locally only:

- browser profile
- output reports
- logs
- temporary exports

## Repository Intention

This public repo packages the method.

Your private project should hold the live data.
